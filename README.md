### 道具规则

1. 每个关卡第4、11、18个红色坦克携带
2. 击中后在16个随机位置生成道具，即使未杀死
3. 道具激活时获得500点
4. 新道具生成时会清除上一个
5. 道具坐标位置（原点为[0, 0]）如下：
   ```
   [3, 3] [3, 9] [3, 15] [3, 21]
   [9, 3] [9, 9] [9, 15] [9, 21]
   [15, 3] [15, 9] [15, 15] [15, 21]
   [21, 3] [21, 9] [21, 15] [21, 21]
   ```
6. 规律：由3, 9, 15, 21这4个横向与4个纵向组合的16个坐标
